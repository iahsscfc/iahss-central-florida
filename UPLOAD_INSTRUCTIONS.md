# Upload Instructions

1. Download and extract the ZIP.
2. Open the extracted `IAHSS_CFC_Website_Master_v1.1` folder.
3. Upload **all files and folders inside it** to the root of the GitHub repository.
4. Allow GitHub to replace matching files.
5. Commit directly to the `main` branch.
6. Wait for the GitHub Pages deployment to show a green checkmark under **Actions**.
7. Open the website and press `Ctrl + F5` once.

Do not upload the outer folder itself.
